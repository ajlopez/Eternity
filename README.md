# Eternity

Eternity II-like puzzle, board, pieces, and solver. Written in JavaScript, for browser and Node.js.

Eternity II is a 256 piece edge-matching puzzle with a prize of $2,000,000 US for the first solution submitted before 31 December 2010.

## Installation

Via npm on Node:

```
npm install eternity
```


## Usage

Reference in your program:

```js
var eternity = require('eternity');
```
