# Eternity

Eternity II-like puzzle, board, pieces, and solver. Written in JavaScript, for browser and Node.js.

## Installation

Via npm on Node:

```
npm install eternity
```


## Usage

Reference in your program:

```js
var eternity = require('eternity');
```
